# Homebrew Tap

Personal Homebrew tap for packages published from my projects.

## Usage

Tap the repository:

```sh
brew tap jcyamacho/tap
```

Install packages by name:

```sh
brew install <name>
```

Some projects may also publish casks:

```sh
brew install --cask <name>
```

## Maintenance

Formulae and casks in this tap are generated or updated by each project's release process. This README intentionally avoids listing packages so it does not need to change when packages are added, removed, or released.
